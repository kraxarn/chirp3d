# chirp3d

Modern and lightweight 3D-focused game engine for Linux/Android through Vulkan, macOS/iOS through Metal and Windows
through DirectX 12, written in C23.

# compiler support

Any compiler with C23 support should work, including clang 19+ (recommended) or gcc 15+.
MSVC is currently not supported, as it doesn't support C23 yet.
